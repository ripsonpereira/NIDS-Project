About us

This project could be used by companies to analyse and examine their network flows to find out if the network had been under attack

This is a Research based project for identifying a suitable model for detecting and classifying the anomalies in the netowork data

Steps to use

  • upload a csv file consisting of one or more rows of packet data
  • click on show data to see the uploaded data or click on show prediction to predict the labels for the uploaded rows of data packets
  • analyse the graphs to check the percentage and count of the anomalus packets in the data
  • review your uploaded data and check the predicted labels for each column

    • Dataset used

    We used the latest available network data from canadian institute of cyber security which is collected by capturing network flows using wireshark packet capture tool for a series of days simulating real life enviornment this captured data was converted into xml which was further converted to csv files for our use. The data set countained roughly 62 lakh rows and 80 columns and 11 different classification labels

    From our research we found that LSTM neural networks was the most reliable model for detecting and clasifying network anomalies
